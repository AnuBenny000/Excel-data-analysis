# Excel-data-analysis
This project focuses on analyzing structured datasets using Microsoft Excel to generate meaningful insights and interactive dashboards.
Key Features:
- Data cleaning and preprocessing
- Use of Pivot Tables and Pivot Charts
- Interactive dashboard creation
- KPI tracking and performance analysis
- Visual representation using charts and graphs

  Tools Used:
- Microsoft Excel
- Pivot Tables
- Charts & Graphs
- Data Filtering & Sorting
- Basic Formulas and Functions

  Outcome:
Generated clear, data-driven insights to support better business decision-making.
